Configuration > GV Coupons

Default Order Status For Zero Balance Orders

Key: DEFAULT_ZERO_BALANCE_ORDERS_STATUS_ID
Path: Configuration > GV Coupons
Description: When an order's balance is zero, this order status will be assigned to it.

Length of the redeem code

Key: SECURITY_CODE_LENGTH
Path: Configuration > GV Coupons
Description: Enter the length of the redeem code
The longer the more secure

New Signup Discount Coupon ID#

Key: NEW_SIGNUP_DISCOUNT_COUPON
Path: Configuration > GV Coupons
Description: Select the coupon

New Signup Gift Voucher Amount

Key: NEW_SIGNUP_GIFT_VOUCHER_AMOUNT
Path: Configuration > GV Coupons
Description: Leave blank for none
Or enter an amount ie. 10 for $10.00

Maximum Discount Coupons Per Page

Key: MAX_DISPLAY_SEARCH_RESULTS_DISCOUNT_COUPONS
Path: Configuration > GV Coupons
Description: Number of Discount Coupons to list per Page

Maximum Discount Coupon Report Results Per Page

Key: MAX_DISPLAY_SEARCH_RESULTS_DISCOUNT_COUPONS_REPORTS
Path: Configuration > GV Coupons
Description: Number of Discount Coupons to list on Reports Page
